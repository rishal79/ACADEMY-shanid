JavaScript String and Math Object Functions

String Functions

The String object in JavaScript provides a range of methods for manipulating and interacting with text strings. Here are some common methods:

1. String.prototype.charAt()

This method returns the character at a specified index in a string.

let str = "Hello";
console.log(str.charAt(1)); // Output: "e"

2. String.prototype.includes()

This method checks if a given substring exists within the string.

let str = "JavaScript is awesome";
console.log(str.includes("awesome")); // Output: true

3. String.prototype.toLowerCase() and String.prototype.toUpperCase()

These methods convert the string to lower case or upper case, respectively.

let str = "Hello World";
console.log(str.toLowerCase()); // Output: "hello world"
console.log(str.toUpperCase()); // Output: "HELLO WORLD"

4. String.prototype.trim()

This method removes whitespace from both ends of a string.

let str = " Hello World! ";
console.log(str.trim()); // Output: "Hello World!"

Math Functions

The Math object in JavaScript provides mathematical constants and functions. Here are some commonly used methods:

1. Math.round()

This method rounds a number to the nearest integer.

let num = 4.7;
console.log(Math.round(num)); // Output: 5

2. Math.floor() and Math.ceil()

These methods round a number down (floor) or up (ceil) to the nearest integer.

let num = 4.7;
console.log(Math.floor(num)); // Output: 4
console.log(Math.ceil(num)); // Output: 5
